You need to create a file named ".env" here

He will contain only one line:

YOUTUBE_KEY=...

In place of ... , you need to copy paste a valid Youtube API key.

_____________


To create a Youtube API key, you can try those steps (or look a tutorial on the internet) :

- Go to Google Developers Console :
https://console.cloud.google.com/apis/credentials?pli=1

- Create a new project.

- On the new project dashboard, click Explore & Enable APIs.

- In the library, navigate to YouTube Data API v3 under YouTube APIs.

- Enable the API.

- Create a credential.

- A screen will appear with the API key.
